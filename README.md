# 52in52 — Building 52 Projects in 52 Weeks (2026)

## Overview
52in52 is a year-long coding challenge where I am building **52 real software projects in 52 weeks** to strengthen my skills in:

- Python & Java
- Object-Oriented Programming
- Data Structures & Algorithms
- APIs, Web Scraping, Databases
- GUIs, Backend Development, and Deployment

This challenge documents my growth from a **first-year Computer Science student** to a confident application builder.
